# Release Notes v0.1.0

## Summary
v0.1.0 provides a validation only FSM core with declarative definitions, invariant checks, and reproducible validation outputs.

## Highlights

- Declarative FSM definition format with structure validation and invariants.
- CLI validator with schema and strict validation modes.
- JSON Schema and API documentation for the definition format.
- Audit trail model and JSONL example outputs.
- CI workflows for tests, linting, and validation checks.
